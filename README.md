# e-Nvernadero Store

Proyecto de e-Commerce de venta de plantas para interior y exterior.

Proyecto realizado con:
 * REACTJS
 * CRA
 * Boostrap
 * Firebase

E-Commerce de compra de plantas de interiores y exteriores.
